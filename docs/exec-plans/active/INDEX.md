## 운영 규칙

- 현재 작업과 직접 관련 없는 active plan은 읽지 않는다.
- 새 상위 문서 작업은 관련 active plan을 확인하거나 새로 만든 뒤 시작한다.
- 완료된 plan은 `completed/`로 이동한다.

# Combat Controls And View

## 목표

## 설계 메모
- 전투 핵심이 패링, 무력화, 캔슬, 위치 조정이므로 자유 시점보다 쿼터뷰 고정 또는 반고정 시점이 맞다.
- 서버는 3D 렌더링 여부와 무관하게 `XZ` 평면 기반 2D 스킬 판정을 유지하는 편이 일관성과 재현성에 유리하다.
- `WASD` 이동과 다수 스킬 사용은 충분히 가능하며, 대신 모든 스킬을 정밀 조준형으로 만들지 않고 즉발/전방/범위 지정/돌진 계열을 섞어야 한다.
- 입력 체계는 왼손 이동, 오른손 조준과 패링을 분리해 패턴 대응 스트레스를 줄이는 방향을 우선한다.
- `python tools/doc_check.py` 기준 내부 링크와 exec-plan 형식 검증은 이번 변경에서 통과했다.

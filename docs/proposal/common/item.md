# 아이템 및 장비 시스템

## 목적

아이템 파밍, 장비 유지, 거래 제한을 통해 무기 기반 올인 빌드와 장기적인 아이템 가치를 함께 유지한다.

## 핵심 규칙

- 모든 전리품은 개인 드롭을 기본으로 한다.
- 장착 무기 타입 기준 스마트 드롭으로 원하는 빌드의 파밍 효율을 보정한다.
- 아이템 등급은 `노멀 -> 언커먼 -> 레어 -> 에픽` 순으로 가치가 올라간다.
- 무기는 랜덤 옵션을 가지며, 전투 핵심 능력인 무력화, 패링 보정도 옵션 후보에 포함된다.
- 재련은 한 번에 한 옵션만 다시 굴리는 방식으로 제한한다.
- 재련을 시도한 아이템은 계정 귀속이 된다.
- 거래 가능 횟수가 0이 되면 타인 거래는 막히지만 계정 공유 창고 이동은 허용한다.
- 사망 시 장착 무기의 내구도가 감소하고, 내구도 0인 무기는 옵션과 전투 성능이 비활성화된다.

## 주요 흐름

### 파밍과 성장

1. 던전 보상으로 개인 드롭 아이템을 획득한다.
2. 장착 무기 타입에 맞는 장비와 스킬북 드롭이 우선 보정된다.
3. 높은 등급 아이템일수록 더 많은 랜덤 옵션과 상위 가치를 가진다.
4. 필요한 경우 재련으로 특정 옵션 하나를 다시 굴린다.

### 거래와 귀속

1. 아이템은 정해진 거래 가능 횟수를 가진다.
2. 유저 간 거래가 성사될 때마다 횟수가 차감된다.
3. 거래 가능 횟수가 0이 되면 계정 귀속 상태가 된다.
4. 계정 귀속 이후에도 같은 계정 내 캐릭터 간 이동은 공유 창고로 처리한다.

### 장비 유지

1. 캐릭터 사망 시 장착 무기 내구도가 즉시 감소한다.
2. 내구도 0 장비는 파괴되지 않지만 전투 성능이 정지된다.
3. 마을에서 골드를 지불해 내구도를 복구한다.

## 제약과 예외

- 스마트 드롭은 장착 무기 기준 드롭 확률을 50% 이상 보정하는 방향으로 설계한다.
- 재련 비용은 같은 아이템 반복 시도 시 빠르게 증가한다.
- 소모품 슬롯은 2개이며 각 슬롯당 최대 5개만 장착한다.
- 던전 내부에서는 소모품 슬롯 구성을 바꾸지 않는다.
- `OptionID + Value` 같은 전송/저장 구조는 구현 문서에서 확정한다.

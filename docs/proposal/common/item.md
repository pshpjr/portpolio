# 아이템 및 장비 시스템

## 목적

아이템 파밍, 장비 유지, 거래 제한을 통해 무기 전환이 가능한 구조 안에서도 주력 무기 중심의 성장 가치를 유지한다.

## 핵심 규칙

- 모든 전리품은 개인 드롭을 기본으로 한다.
- 던전 입장 시 선택한 전리품 선호 무기 기준의 soft bias로 파밍 효율을 보정한다.
- 아이템 등급은 `노멀 -> 언커먼 -> 레어 -> 에픽` 순으로 가치가 올라간다.
- 무기는 랜덤 옵션을 가지며, 전투 핵심 능력인 무력화, 패링 보정도 옵션 후보에 포함된다.
- 재련은 한 번에 한 옵션만 다시 굴리는 방식으로 제한한다.
- 재련을 시도한 아이템은 계정 귀속이 된다.
- 거래 가능 횟수가 0이 되면 타인 거래는 막히지만 계정 공유 창고 이동은 허용한다.
- 사망 시 장착 무기의 내구도가 감소하고, 내구도 0인 무기는 옵션과 전투 성능이 비활성화된다.

## 주요 흐름

### 파밍과 성장

1. 던전 보상으로 개인 드롭 아이템을 획득한다.
2. 입장 시 선택한 전리품 선호 무기에 맞는 무기/스킬북 드롭이 우선 보정된다.
3. 높은 등급 아이템일수록 더 많은 랜덤 옵션과 상위 가치를 가진다.
4. 필요한 경우 재련으로 특정 옵션 하나를 다시 굴린다.

### 거래와 귀속

1. 아이템은 정해진 거래 가능 횟수를 가진다.
2. 유저 간 거래가 성사될 때마다 횟수가 차감된다.
3. 거래 가능 횟수가 0이 되면 계정 귀속 상태가 된다.
4. 계정 귀속 이후에도 같은 계정 내 캐릭터 간 이동은 공유 창고로 처리한다.

### 장비 유지

1. 캐릭터 사망 시 장착 무기 내구도가 즉시 감소한다.
2. 내구도 0 장비는 파괴되지 않지만 전투 성능이 정지된다.
3. 마을에서 골드를 지불해 내구도를 복구한다.

## 아이템 등급별 옵션 개수

| 등급 | 무기 옵션 수 | 방어구/장신구 옵션 수 |
|---|---|---|
| 노멀 | 0 | 0 |
| 언커먼 | 1 | 1 |
| 레어 | 2 | 2 |
| 에픽 | 3 | 3 |

## 거래 가능 횟수 (등급별)

| 등급 | 초기 거래 가능 횟수 |
|---|---:|
| 노멀 | 5 |
| 언커먼 | 3 |
| 레어 | 2 |
| 에픽 | 1 |

- 유저 간 거래가 성사될 때마다 양도받은 쪽의 아이템 거래 가능 횟수가 1 차감된다.
- 재련 시도 시 계정 귀속이 되며, 계정 귀속 후에도 거래 가능 횟수 자체는 유지된다 (이미 귀속이므로 의미 없음).
- 재련 실패 시에도 계정 귀속은 동일하게 적용된다.

## 소모품 종류

| 소모품 | 효과 | 쿨다운 |
|---|---|---|
| 회복 영약 (소) | HP +200 즉시 회복 | 30초 |
| 회복 영약 (대) | HP +500 즉시 회복 | 45초 |
| 투혼 향유 | AttackPower +15%, 30초 지속 | 120초 |
| 석갑 향유 | DamageReduction +0.10, 30초 지속 | 120초 |

- 소모품 슬롯은 2개, 슬롯당 최대 5개 보유.
- 소모품 쿨다운은 아이템 단위가 아닌 슬롯 단위로 적용된다.
- NPC 상점에서 회복 영약(소)만 구매 가능하다. 나머지는 던전 드롭.

## 방어구/장신구 슬롯 구성

| 슬롯 | 분류 |
|---|---|
| 무기 | 1개 |
| 상의 (방어구) | 1개 |
| 하의 (방어구) | 1개 |
| 장갑 (방어구) | 1개 |
| 목걸이 (장신구) | 1개 |
| 반지 (장신구) | 2개 |

- v1 구현 기준 방어구/장신구의 방어 스탯은 `Defense`, `MaxHP` 위주다.
- 방어구와 장신구의 드롭은 무기와 동일하게 스마트 드롭 보정을 받지 않는다.

## 제약과 예외

- 스마트 드롭은 원하는 무기 파밍이 체감되되, 오프빌드와 공유 창고 활용 여지는 남는 수준으로만 보정한다.
- 재련 비용은 같은 아이템 반복 시도 시 빠르게 증가한다.
- 소모품 슬롯은 2개이며 각 슬롯당 최대 5개만 장착한다.
- 던전 내부에서는 소모품 슬롯 구성을 바꾸지 않는다.
- `OptionID + Value` 같은 전송/저장 구조는 구현 문서에서 확정한다.
- `정찰 원정` 전용 보상은 연습용/저가치 보상으로 묶고, NPC 판매 가치가 없는 아이템을 포함할 수 있다.
- UI에는 `거래 가능 횟수`, `계정 귀속`, `연습 보상` 상태를 아이콘과 툴팁으로 함께 노출한다.

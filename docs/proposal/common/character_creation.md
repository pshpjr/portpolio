# 캐릭터 생성

## 목적

새 캐릭터 생성 시 시작 무기 선택, 초기 상태, 지급 아이템을 정의한다.

## 핵심 규칙

- 계정당 최대 3캐릭터를 생성할 수 있다.
- 캐릭터 이름은 영문+한글 2~12자, 중복 불가.
- 생성 시 시작 무기 타입(검방/대검/지팡이)을 하나 선택한다.
- 현재 장착 무기 타입이 스킬 세트와 아이덴티티 게이지를 결정한다.
- 캐릭터는 생성 후에도 마을에서 다른 무기 타입을 장착할 수 있다.
- 레벨 1 초기 스탯은 `combat_stats.md`의 v1 플레이어 기본값을 따른다.

## 주요 흐름

1. 이름 입력 (서버에서 중복 검사 수행)
2. 시작 무기 타입 선택 (각 타입 특성 간략 설명 표시)
3. 최종 확인
4. DB 저장 및 캐릭터 목록으로 복귀

## 초기 지급 아이템

| 항목 | 내용 |
|---|---|
| 기본 무기 | 선택 무기 타입의 기본 무기 1개 (강화 0, 랜덤 옵션 없음) |
| HP 회복 포션 | 소모품 슬롯용 회복 영약(소) 5개 |
| 골드 | 500 골드 |
| 기본 스킬 | 선택 무기의 기본 스킬 4종 자동 해금 (`Q/E/R/F`) |

## 무기 타입별 특성 요약

선택 화면 표시용 간략 정보다.

| 무기 | 역할 | 특성 |
|---|---|---|
| 검방 | 전위 방어 | 패링 보정 +0.04초, 위협도 높음 |
| 대검 | 고위력 딜 | 무력화 집중, 단일 강타 |
| 지팡이 | 원거리 제어 | 속성 활용, 범위 제어 |

## 제약과 예외

- 캐릭터 삭제 기능은 포트폴리오 범위 밖이다.
- 외형 커스터마이징은 현재 범위 밖이다.
- 다른 무기 타입 전환은 마을에서만 허용하며, 던전 입장 후에는 현재 무기 구성이 잠긴다.

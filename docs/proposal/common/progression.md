# 경험치와 레벨업

## 목적

경험치 획득, 레벨업 요구치, 레벨업 보상을 정의한다.

## 핵심 규칙

- 최대 레벨은 40이다.
- 경험치는 캐릭터 단위로 누적한다.
- 파티 플레이 시 경험치는 참여 인원에게 균등 분배한다.
- 레벨업 시 스킬 포인트 1점을 지급한다.
- 스킬 포인트는 스킬 레벨 업그레이드와 변형 선택에 사용한다.

## 경험치 획득원

| 획득원 | 경험치 기준 |
|---|---|
| 일반 몬스터 처치 | 캐릭터 레벨 기준 동급 몬스터 = 15 EXP |
| 정예 몬스터 처치 | 동급 기준 50 EXP |
| 보스 처치 | 동급 기준 300 EXP |
| 던전 클리어 보너스 | 100 EXP (제한형 모드만) |
| 기믹 성공 보너스 | 30 EXP |

## 레벨 구간별 요구 EXP

| 구간 | 설명 |
|---|---|
| 레벨 1~10 | 각 레벨업 요구치: 300, 500, 750, 1050, 1400, 1800, 2250, 2750, 3300 (누적 성장) |
| 레벨 11~25 | 레벨당 약 4000~8000 (완만한 선형 증가) |
| 레벨 26~40 | 레벨당 약 9000~15000 (후반 완만화) |

- 기준: 동급 던전 1회 클리어(일반 몬스터 모두 처치 + 보스) 시 약 1.5~2 레벨 상승 속도 목표.

## 레벨업 보상

- 스킬 포인트 +1 (매 레벨).
- 스탯 자동 증가 (`combat_stats.md`의 레벨당 증가량 적용).
- 특정 레벨 도달 시 추가 보상:
  - 레벨 10: 두 번째 스킬 슬롯 개방
  - 레벨 20: 세 번째 스킬 슬롯 개방

## 스킬 포인트 사용 규칙

- 스킬 레벨은 최대 5까지 올릴 수 있다.
- 레벨 3에서 변형(트라이포드) 선택이 가능하다.
- 레벨 5에서 두 번째 변형 선택이 가능하다.
- 스킬 포인트 환불은 마을 특정 NPC에서 골드 소모 후 가능하다.

## 제약과 예외

- 레벨 제한 이상 몬스터 처치 시 경험치는 동급 기준값을 사용한다.
- 파티 EXP 보너스(협동 보너스)는 현재 범위 밖이다.
- 정확한 EXP 수치는 플레이 테스트 후 조정한다.

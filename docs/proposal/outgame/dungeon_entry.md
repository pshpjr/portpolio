# 던전 입장 시스템

## 1. 개요
파티장이 던전 입구(포탈)에서 입장 요청을 하면, 파티원 전원의 동의를 거쳐 인스턴스가 생성되고 던전으로 이동한다.

## 2. 던전 입장 모드 선택 (Rewards Mode Selection)
파티장은 입장 요청 시 다음 두 가지 보상 모드 중 하나를 선택해야 한다.
- **제한형 보상 모드 (Limited Main Reward Mode)**:
  - **성격**: 일일/주간 보상 획득 횟수(예: 캐릭터당 1일 3회)를 소모하는 정규 공략 모드.
  - **보상**: 에픽/레어 등급 장비 및 희귀 스킬북 획득 가능.
  - **제한**: 일일 횟수를 모두 소모한 유저는 해당 모드로 입장할 수 없다.
- **무제한형 보상 모드 (Unlimited Practice Mode)**:
  - **성격**: 보상 획득 횟수 제한 없이 무한히 입장 가능한 숙련 및 재료 파밍 모드.
  - **보상**: 기본 재료 및 소량의 경험치/골드만 지급되며, 고등급 아이템은 드랍되지 않는다.
  - **의도**: 유저의 숙련도 향상 및 특정 무기 연습을 지원하며, 콘텐츠 소모 속도를 조절한다.

## 3. 입장 프로세스 및 동의 (Dungeon Entry Consent)
- **동의 요청**: 파티장이 모드를 선택하고 입장 요청 시, 모든 파티원에게 동의 팝업(30초)이 전송된다.
- **전원 동의 필수**: 파티원 중 한 명이라도 거절하거나 보상 횟수가 부족할 경우 입장이 취소된다.
- **인스턴스 생성**: 전원 동의 시 **인던 전용 싱글 스레드**에 인스턴스가 할당되어 던전이 시작된다.

## 4. 인원수 기반 기믹 스케일링
- 입장 시 파티 인원(1~4인)을 체크하여, 해당 인원수에 최적화된 기믹 개수와 몬스터 체력이 설정된다.
- 1인 플레이 시 자동으로 기믹 난이도가 하향 조정되어 솔로 플레이가 가능하다.

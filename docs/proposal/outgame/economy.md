# 경제 시스템

## 목적

골드 싱크와 거래 제한을 통해 아이템 가치와 장기 파밍 동기를 유지한다.

## 핵심 규칙

- 경제의 핵심은 골드 소모처와 아이템 가치 보존이다.
- NPC 상점은 소모품과 기초 스킬북 공급처이자 불필요한 아이템 매각처다.
- 장비 수리와 옵션 재련은 주요 골드 싱크다.
- 유저 간 거래는 마을에서만 허용하며 거래 가능 횟수를 소모한다.
- 거래 가능 횟수가 끝난 아이템은 계정 귀속으로 전환된다.

## 주요 흐름

### NPC 상점

1. 플레이어는 골드로 소모품과 기초 등급 스킬북을 구매한다.
2. 필요 없는 아이템은 낮은 비율의 가격으로 판매한다.
3. 상점 재고는 기본적으로 제한하지 않는다.

### 수리와 재련

1. 장비 손상 상태에 따라 수리 비용을 지불한다.
2. 반복 재련 시 비용이 빠르게 증가한다.
3. 필요하면 특수 재화를 추가로 소모하는 구조를 붙일 수 있다.

### 유저 간 거래

1. 두 플레이어가 모두 마을에 있을 때만 거래를 시작한다.
2. 거래 가능한 아이템과 골드를 제시한다.
3. 양측 확인 후 서버가 원자적으로 교환한다.
4. 거래가 끝나면 해당 아이템의 거래 가능 횟수를 차감한다.

## 골드 획득 경로

| 획득원 | 골드 기준 |
|---|---|
| 일반 몬스터 처치 | 캐릭터 레벨 기준 1~3 골드 |
| 정예 몬스터 처치 | 5~15 골드 |
| 보스 처치 | 50~100 골드 |
| 아이템 NPC 판매 | 아이템 등급 기준 10~500 골드 |
| 유저 간 거래 | 협의 금액 (수수료 5% 제외) |

## 수리비 기준

| 무기 등급 | 내구도 1당 수리비 |
|---|---|
| 노멀 | 5 골드 |
| 언커먼 | 12 골드 |
| 레어 | 30 골드 |
| 에픽 | 75 골드 |

- 최대 내구도는 무기 등급에 무관하게 30으로 통일한다.
- 전손(0) 상태 수리는 동일 단가를 적용한다.

## 재련비 기준

| 재련 시도 횟수 (동일 아이템) | 비용 |
|---|---|
| 1회 | 200 골드 |
| 2회 | 500 골드 |
| 3회 | 1000 골드 |
| 4회 이상 | 이전 비용 × 1.8 (올림) |

- 재련 비용은 아이템별로 누적 관리한다.
- 다른 옵션을 굴리는 경우에도 누적 횟수에 포함된다.

## 제약과 예외

- 중급 이상 스킬북은 상점이 아니라 던전 드롭 중심으로 유지한다.
- 판매 불가 아이템에는 캐릭터 귀속, 계정 귀속, 거래 횟수 0 아이템이 포함된다.
- 거래에는 소정의 수수료를 붙여 추가 골드 싱크로 사용한다.
- 최대 골드 보유량 같은 수치는 구현 단계에서 조정 가능하다.

# 사망, 부활, 전멸 처리

## 목적

캐릭터 사망, 체크포인트, 부활, 전멸 처리 규칙을 정의한다.

## 핵심 규칙

- HP가 0이 되면 즉시 사망 상태로 전환된다.
- 사망 시 장착 무기의 내구도가 5 감소한다.
- 내구도 0인 무기는 전투 성능(스킬, 아이덴티티 효과)이 비활성화된다.
- 던전 내 체크포인트는 구역 진입 시 자동 저장된다.
- 체크포인트에서 부활하면 HP 50%로 복귀한다.
- 파티원이 부활 스킬로 현장 부활시킬 수 있다 (HP 30% 복귀).
- 파티 전원 사망 시 전멸 처리된다.
- 현장 부활 또는 체크포인트 부활 직후에는 짧은 안전 보호를 준다.

## 주요 흐름

1. HP 0 -> 사망 상태 전환
2. 내구도 감소 처리
3. 현장 부활 가능 여부 판단 (파티원 생존 여부, 부활 쿨다운)
4. 현장 부활 없을 시 체크포인트 부활 선택
5. 전멸 시 던전 실패 처리

## 상세 규칙

| 항목 | 값 |
|---|---|
| 사망 시 내구도 감소 | -5 |
| 내구도 최솟값 | 0 (파괴 없음) |
| 체크포인트 부활 HP | 최대 HP의 50% |
| 현장 부활 HP | 최대 HP의 30% |
| 현장 부활 시전 시간 | 3초 (이동 시 취소) |
| 현장 부활 쿨다운 | 60초 (부활 대상 기준) |
| 부활 직후 보호 | 2초간 피해 50% 감소 |
| 전멸 후 대기 시간 | 5초 후 던전 실패 화면 |

## 전멸 처리

- 전멸 시 던전 진행도는 초기화된다.
- 내구도 감소는 전멸 후에도 유지된다.
- 제한형 모드 입장 횟수는 전멸해도 소모된 것으로 처리한다.
- 전멸 시 드롭된 아이템은 소멸한다.

## 회복 가능한 실패와 전멸의 구분

- 파티원 일부 사망은 `회복 가능한 실패`로 취급하고, 현장 부활과 재정렬 시간을 준다.
- 체크포인트 부활 후에는 직전 구역 목표를 다시 설명해 파티가 바로 흐름을 복구하게 한다.
- 모든 파티원이 전투 불능이고 현장 부활 수단이 없을 때만 `전멸`로 판정한다.

## 체크포인트 규칙

- 각 구역 진입 시 해당 구역이 체크포인트로 설정된다.
- 체크포인트 부활 시 현재 구역의 시작 지점에서 부활한다.
- 이미 처치한 몬스터는 부활하지 않는다.

## 제약과 예외

- 현장 부활 스킬의 구체적인 조건(스킬 슬롯 배정 등)은 스킬 데이터 문서에서 확정한다.
- 솔로 플레이 시 현장 부활은 불가능하다.
- 체크포인트 없는 구역(구역 1, 보스 구역)에서 전멸 시 던전 입구부터 재시작한다.

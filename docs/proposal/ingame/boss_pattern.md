# 보스 패턴 시스템 (주술사)

## 목적

최종 보스 `주술사`의 공략 구조, 페이즈 변화, 패턴 대응 원칙을 정의한다.

## 핵심 규칙

- 주술사는 2페이즈 보스로 설계한다.
- 전투의 핵심은 패턴 이해, 패링, 무력화, 속성 약점 활용, 캔슬 대응이다.
- 번개는 약점 속성이고 독은 저항 속성이다.
- 무력화는 큰 딜 타임과 추가 보상 조건을 만든다.
- 일부 시전은 `Heavy` 또는 `Crush` 등급 스킬로만 캔슬 가능하다.
- 8분이 지나면 광폭화가 발동한다.

## 주요 흐름

### 공략 루프

1. 파티는 약점 속성과 역할 분담을 맞춰 전투를 시작한다.
2. 보스의 즉발 패턴과 시전 패턴을 구분해 대응한다.
3. 노란색, 빨간색 시전 바에 맞춰 적절한 캔슬 수단을 넣는다.
4. 약점 속성과 무력화 스킬을 집중해 큰 빈틈을 만든다.
5. 50% 이하에서 2페이즈로 넘어가며 패턴 강도가 높아진다.

### 무력화 루프

1. 패링 성공, 약점 속성 활용, 캔슬 성공으로 무력화 게이지를 줄인다.
2. 게이지가 0이 되면 즉시 현재 패턴을 중단하고 6초 다운 상태로 진입한다.
3. 전투 중 무력화 2회 이상 달성 시 추가 보상 조건을 만족한다.

## 패턴 발동 조건

### Phase 1 패턴

| 패턴 | 발동 조건 | 쿨다운 | 캔슬 등급 |
|---|---|---|---|
| 토템 설치 | 전투 시작 5초 후 최초 발동, 이후 쿨다운 | 30초 | 불가 |
| 영혼 투사체 | 쿨다운 경과 시 패턴 풀에서 선택 | 8초 | Heavy |
| 지팡이 휘두르기 | 근접 거리(4m 이내) 대상 존재 시 우선 발동 | 5초 | Heavy |
| 주술 강화 시전 | HP ≤ 75% 이후 쿨다운 경과 시 | 45초 | Heavy / Crush |
| 암흑 소용돌이 | HP ≤ 50% 직전 1회만 발동 (페이즈 전환 직전 강제 패턴) | 1회 제한 | Crush |

- Phase 1 일반 패턴은 쿨다운이 끝난 패턴 중 조건을 만족하는 패턴을 가중치 기반으로 선택한다.
- 가중치: `영혼 투사체(3)`, `지팡이 휘두르기(조건 충족 시 5, 미충족 시 0)`, `주술 강화 시전(조건 충족 시 2, 미충족 시 0)`.
- 다운 상태 중에는 패턴 선택을 중단하고, 다운 종료 후 3초 뒤 재개한다.

### Phase 2 패턴

| 패턴 | 발동 조건 | 쿨다운 | 캔슬 등급 |
|---|---|---|---|
| 영혼 늑대 소환 | 2페이즈 진입 직후 자동 발동 (1회 고정) | 1회 | 불가 |
| 주술 버프 | 영혼 늑대 1마리 이상 생존 중, 쿨다운 경과 시 | 20초 | Heavy |
| 추적 영혼구 | 쿨다운 경과 시 패턴 풀에서 선택 | 12초 | Heavy |
| 격노 주술 | HP ≤ 30% 이후 쿨다운 경과 시 | 60초 | Crush |
| 영혼 폭풍 | HP ≤ 20% 최초 1회 발동, 이후 쿨다운 | 90초 | 불가 |

- Phase 2 가중치: `추적 영혼구(4)`, `주술 버프(조건 충족 시 3, 미충족 시 0)`, `격노 주술(조건 충족 시 2, 미충족 시 0)`, `영혼 폭풍(조건 충족 시 2, 미충족 시 0)`.

## 패턴 요약

### Phase 1

- 토템 설치: 장판 압박용 즉발 패턴, 캔슬 불가
- 영혼 투사체: 패링 성공 시 무력화 기여
- 지팡이 휘두르기: 근접 광역, 패링 시 큰 경직 유도
- 주술 강화 시전: 노란 시전 바, `Heavy` 또는 `Crush`로 캔슬
- 암흑 소용돌이: 빨간 시전 바, `Crush`만 확정 캔슬

### Phase 2

- 영혼 늑대 소환: 시작 압박용 소환 패턴
- 주술 버프: 소환 몬스터 강화
- 추적 영혼구: 회피 중심 대응 패턴
- 격노 주술: 빨간 시전 바, 캔슬 실패 시 30초간 임시 광폭화 상태 진입
- 영혼 폭풍: 캔슬 불가 전면 압박 패턴

## 구현용 보스 수치

| 항목 | 값 |
|---|---:|
| `MaxHP` | 18000 |
| `Defense` | 220 |
| `StaggerGauge` | 1000 |
| `StaggerResistance` | 180 |
| `StatusResistance` | 140 |
| `EnrageTime` | 480초 |
| `Phase2Threshold` | HP 50% |

## 광폭화 (Enrage) 효과

- 전투 시작 480초(8분) 경과 시 자동 발동.
- 격노 주술 캔슬 실패 시에도 30초간 임시 광폭화 상태 진입 (타이머와 무관하게 즉시 적용, 30초 후 해제).

| 광폭화 효과 | 수치 |
|---|---|
| 공격력 배율 | ×1.5 |
| 모든 패턴 쿨다운 | ×0.5 (절반으로 감소) |
| 시전 바 캔슬 | 불가 (노란/빨간 모두 캔슬 무효화) |

- 영구 광폭화(타이머 만료)는 해제되지 않는다.
- 임시 광폭화(격노 주술 실패)는 30초 후 해제되며, 이후 영구 광폭화 타이머는 계속 진행된다.

## 무력화 후처리

| 조건 | 효과 |
|---|---|
| 무력화 성공 즉시 | 현재 시전/이동 취소 |
| 다운 상태 6초 | 받는 피해 `+20%`, 이동/회전/시전 불가 |
| 다운 종료 직후 8초 | 재무력화 불가 보호 상태 |
| 두 번째 무력화부터 | `StaggerResistance +40` |
| 세 번째 이상 | 추가 회차마다 `+25`, 최대 누적 `+120` |

### StaggerResistance 누적 초기화 조건

- `StaggerResistance` 누적은 **페이즈 전환 시에만 초기화**된다.
- Phase 1 → Phase 2 전환 시 `StaggerResistance`가 기본값(180)으로 리셋된다.
- 다운 후 보호 상태 해제 시에는 초기화되지 않는다 (전투 누적 유지).

## 강제 타게팅 규칙

| 패턴 | 대상 선택 |
|---|---|
| `추적 영혼구` | 현재 최원거리 플레이어 고정 |
| `영혼 늑대 소환 직후 돌진` | 현재 최저 HP 플레이어 고정 |
| `격노 주술` 준비 동작 | 현재 어그로 1위 고정 |

- 강제 타게팅 중에도 위협도 누적은 계속한다.
- 강제 타게팅 종료 후 즉시 위협도 1위 대상을 다시 계산한다.

## 제약과 예외

- 무력화 게이지 수치, 지속 시간, 누적 저항 값은 이 문서를 기본값으로 구현하고, 추후 플레이 로그로 미세 조정한다.
- 약점 속성은 초반에 직접 공개하지 않고 전투 연출로 유추하게 한다.
- 소환 그룹의 DB/런타임 구조 같은 구현 세부는 별도 구현 문서에서 확정한다.
- 페이즈와 패턴 수를 유지하되, 세부 숫자와 효과량은 후속 조정 가능하다.

# 보스 패턴 시스템 (주술사)

## 목적

최종 보스 `주술사`의 공략 구조, 페이즈 변화, 패턴 대응 원칙을 정의한다.

## 핵심 규칙

- 주술사는 2페이즈 보스로 설계한다.
- 전투의 핵심은 패턴 이해, 패링, 무력화, 속성 약점 활용, 캔슬 대응이다.
- 번개는 약점 속성이고 독은 저항 속성이다.
- 무력화는 큰 딜 타임과 추가 보상 조건을 만든다.
- 일부 시전은 `Heavy` 또는 `Crush` 등급 스킬로만 캔슬 가능하다.
- 8분이 지나면 광폭화가 발동한다.

## 주요 흐름

### 공략 루프

1. 파티는 약점 속성과 역할 분담을 맞춰 전투를 시작한다.
2. 보스의 즉발 패턴과 시전 패턴을 구분해 대응한다.
3. 노란색, 빨간색 시전 바에 맞춰 적절한 캔슬 수단을 넣는다.
4. 약점 속성과 무력화 스킬을 집중해 큰 빈틈을 만든다.
5. 50% 이하에서 2페이즈로 넘어가며 패턴 강도가 높아진다.

### 무력화 루프

1. 패링 성공, 약점 속성 활용, 캔슬 성공으로 무력화 게이지를 줄인다.
2. 게이지가 0이 되면 짧은 시간 동안 집중 딜 기회를 얻는다.
3. 전투 중 무력화 2회 이상 달성 시 추가 보상 조건을 만족한다.

## 패턴 요약

### Phase 1

- 토템 설치: 장판 압박용 즉발 패턴, 캔슬 불가
- 영혼 투사체: 패링 성공 시 무력화 기여
- 지팡이 휘두르기: 근접 광역, 패링 시 큰 경직 유도
- 주술 강화 시전: 노란 시전 바, `Heavy` 또는 `Crush`로 캔슬
- 암흑 소용돌이: 빨간 시전 바, `Crush`만 확정 캔슬

### Phase 2

- 영혼 늑대 소환: 시작 압박용 소환 패턴
- 주술 버프: 소환 몬스터 강화
- 추적 영혼구: 회피 중심 대응 패턴
- 격노 주술: 빨간 시전 바, 실패 시 사실상 광폭화 조기 진입
- 영혼 폭풍: 캔슬 불가 전면 압박 패턴

## 제약과 예외

- 무력화 게이지 수치, 지속 시간, 누적 저항 값은 현재 기획 기준이며 밸런스 조정 대상이다.
- 약점 속성은 초반에 직접 공개하지 않고 전투 연출로 유추하게 한다.
- 소환 그룹의 DB/런타임 구조 같은 구현 세부는 별도 구현 문서에서 확정한다.
- 페이즈와 패턴 수를 유지하되, 세부 숫자와 효과량은 후속 조정 가능하다.

# 보스 패턴 시스템 (혼식 주술사)

## 목적

최종 보스 `혼식 주술사`의 공략 구조, 핵심 패턴, 대응 원칙을 정의한다.

## 판타지 요약

`혼식 주술사`는 성소의 정령을 제물로 삼아 의식을 증폭시키는 보스다. 플레이어는 시전을 끊고, 패링으로 빈틈을 만들고, 무력화 구간에 화력을 집중해 의식을 붕괴시킨다.

## 핵심 규칙

- v1 보스전의 핵심은 `패턴 이해`, `패링`, `인터럽트`, `무력화 집중`이다.
- v1의 메인 보스 루프는 `읽기 쉬운 1페이즈 중심`으로 구성한다.
- 번개는 약점 속성이고 독은 저항 속성이다.
- 무력화는 큰 딜 타임과 추가 보상 조건을 만든다.
- 핵심 시전은 `인터럽트 가능` 상태로 표시된다.
- 8분이 지나면 광폭화가 발동한다.

## 주요 흐름

### 공략 루프

1. 파티는 약점 속성과 역할 분담을 맞춰 전투를 시작한다.
2. 보스의 즉발 패턴과 시전 패턴을 구분해 대응한다.
3. 인터럽트 가능한 시전 바가 열리면 적절한 스킬을 넣는다.
4. 약점 속성과 무력화 스킬을 집중해 큰 빈틈을 만든다.
5. 무력화 다운 구간에 핵심 화력을 집중한다.

| 패턴 | 형상 | 범위 | 전조 | 전조 시간 | 플레이어 답 | 성공 보상 | 실패 페널티 |
|---|---|---|---|---:|---|---|---|

| 영혼 투사체 | 직선 투사체 | 길이 12m / 폭 1.2m | 보스 정면 시전 바 + 투사체 발광 | 0.8초 | 패링 또는 측면 회피 | 패링 시 무력화 추가 누적 | 연속 피격 시 후열 압박 |

| 주술 강화 시전 | 자기중심 시전 | 반경 5m | 긴 시전 바 + 화면 상단 인터럽트 표기 | 1.8초 | 인터럽트 스킬 집중 | 시전 취소 + 빈틈 발생 | 버프 상태로 일정 시간 공격 강화 |
| 영혼 파열 | 광역 장판 | 반경 5m | 발밑 장판 예고 + 화면 암전 효과 | 1.2초 | 파티 분산 후 재집결 | 회피 성공 시 안전 딜 타임 유지 | 광역 피해와 위치 붕괴 |

## 무력화 루프

1. 패링 성공, 인터럽트 성공, 약점 속성 활용으로 무력화 게이지를 줄인다.
2. 게이지가 0이 되면 즉시 현재 패턴을 중단하고 6초 다운 상태로 진입한다.
3. 전투 중 무력화 2회 이상 달성 시 추가 보상 조건을 만족한다.

### v1 무력화 기여 기본값

| 행동 | 기본 기여 |
|---|---:|
| 패링 성공 | 120 |
| 인터럽트 성공 | 180 |
| 약점 속성 핵심 스킬 적중 | 60 |

## 구현용 보스 수치

| 항목 | 값 |
|---|---:|
| `MaxHP` | 18000 |
| `Defense` | 220 |
| `StaggerGauge` | 1000 |
| `StaggerResistance` | 180 |
| `StatusResistance` | 140 |
| `EnrageTime` | 480초 |
| `InterruptWindow` | 주요 시전마다 1.5~2.0초 |

## 광폭화 (Enrage) 효과

- 전투 시작 480초(8분) 경과 시 자동 발동.

| 광폭화 효과 | 수치 |
|---|---|
| 공격력 배율 | ×1.5 |
| 모든 패턴 쿨다운 | ×0.5 (절반으로 감소) |
| 인터럽트 | 불가 |

- 영구 광폭화(타이머 만료)는 해제되지 않는다.

## 무력화 후처리

| 조건 | 효과 |
|---|---|
| 무력화 성공 즉시 | 현재 시전/이동 취소 |
| 다운 상태 6초 | 받는 피해 `+20%`, 이동/회전/시전 불가 |
| 다운 종료 직후 8초 | 재무력화 불가 보호 상태 |
| 두 번째 무력화부터 | `StaggerResistance +40` |
| 세 번째 이상 | 추가 회차마다 `+25`, 최대 누적 `+120` |

## 제약과 예외

- 무력화 게이지 수치, 지속 시간, 누적 저항 값은 이 문서를 기본값으로 구현하고, 추후 플레이 로그로 미세 조정한다.
- 약점 속성은 초반에 직접 공개하지 않고 전투 연출로 유추하게 한다.
- 소환 그룹의 DB/런타임 구조 같은 구현 세부는 별도 구현 문서에서 확정한다.
- 2페이즈 확장, 소환 압박 강화, 복잡한 강제 타게팅은 후속 확장으로 미룬다.
- v1은 패턴 종류를 적게 유지하는 대신, 전조와 대응이 분명하게 읽히는 것을 우선한다.
- 전조 시간은 현재 기준값이며, 프로토타입 플레이 로그로 조정한다.

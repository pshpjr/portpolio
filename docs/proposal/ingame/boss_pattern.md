# 보스 패턴 시스템 (주술사)

## 1. 개요
최종 보스 '주술사'는 2페이즈로 구성되며, 플레이어의 협동과 패턴 대응 능력을 시험한다.
- **약점 속성**: 번개 (Lightning) — 캔슬 및 무력화 게이지 감소량 +50%
- **저항 속성**: 독 (Poison) — 무력화 게이지 감소량 -50%
- 약점/저항 속성은 전투 초반에 숨겨져 있으며, 번개 공격 시 보스 몸이 푸른빛으로 반짝이는 연출로 플레이어가 유추할 수 있다.

---

## 2. 무력화 게이지 (Stagger Gauge)

| 구분 | 값 |
|-----|---|
| Phase 1 무력화 게이지 | 100 |
| Phase 2 무력화 게이지 | 150 (페이즈 전환 시 초기화) |
| 무력화 지속 시간 | 4초 |
| 누적 저항 증가 | 무력화 1회당 +20% (페이즈 전환 시 초기화) |

---

## 3. Phase 1 (HP 100% ~ 50%)

### 패턴 목록

#### 3-1. 토템 설치 (즉발형 — 캔슬 불가)
- 지속 피해를 주는 장판 토템을 설치한다.
- 유저는 토템을 우선 파괴하거나 장판 밖으로 보스를 유인해야 한다.
- 무력화 게이지 기여: 없음 (토템 파괴로 게이지 감소 없음).

#### 3-2. 영혼 투사체 (즉발형 — 캔슬 불가)
- 부채꼴 모양으로 3발의 투사체를 발사한다.
- 회피 또는 패링으로 대응. 패링 성공 시 무력화 게이지 **-15** 감소.

#### 3-3. 지팡이 휘두르기 (즉발형 — 캔슬 불가)
- 근접한 유저에게 광역 물리 공격.
- 패링 성공 시 무력화 게이지 **-20** + 보스 1.5초 경직.

#### 3-4. 주술 강화 시전 (**노란 시전 바** — 캔슬 가능)
- **시전 시간**: 3초.
- **효과**: 시전 완료 시 자신에게 공격력 +30% 버프 10초 부여.
- **캔슬 방법**:
  - Heavy 스킬 적중 → 50% 확률 캔슬, 무력화 게이지 **-20**
  - Crush 스킬 적중 → 확정 캔슬, 무력화 게이지 **-35**
  - 파티원 2인 Heavy+ 동시 적중 → 확정 캔슬, 게이지 **-합산**
- **캔슬 실패 패널티**: 이후 보스 공격 피해 +30% (버프 지속 동안).

#### 3-5. 암흑 소용돌이 (**빨간 시전 바** — Crush 등급만 캔슬)
- **시전 시간**: 2초.
- **효과**: 광역 암흑 속성 피해 + 주변 5m 내 유저에게 Shadow 스택 2 부여.
- **캔슬 방법**: Crush 스킬만 확정 캔슬 가능. Heavy 스킬로는 캔슬 불가 (게이지만 감소).
- **캔슬 실패 패널티**: Shadow 스택이 3이 된 유저는 Blind 상태이상 발동.

---

## 4. Phase 2 (HP 50% ~ 0%)

Phase 1의 패턴이 강화되며 새로운 패턴이 추가된다.

#### 4-1. 영혼 늑대 소환 (즉발형 — 캔슬 불가)
- 진입 시 강력한 근접 소환수 2마리를 소환.
- 소환수는 무력화 게이지 없음 (집중 화력으로 제거).

#### 4-2. 주술 버프 (즉발형 — 캔슬 불가)
- 소환된 몬스터들의 공격력 대폭 강화.
- 무력화 게이지 기여: 없음.

#### 4-3. 추적 영혼구 (즉발형 — 캔슬 불가)
- 대상을 끝까지 추격하는 유도 투사체.
- 패링 불가 — 대쉬로 회피. 적중 시 Fire 스택 2 부여.

#### 4-4. 격노 주술 (**빨간 시전 바** — Crush 등급만 캔슬)
- **시전 시간**: 4초.
- **효과**: 완료 시 자신에게 Enrage 버프 (전 능력치 +50%, 15초).
- **캔슬 방법**: Crush 스킬 확정 캔슬. 무력화 게이지 **-40**.
- **캔슬 실패 패널티**: 광폭화(Enrage) 조기 발동으로 패턴이 극단적으로 강화.
- **전략 포인트**: 이 패턴은 무력화 직후 재사용되지 않으므로, 팀이 무력화를 맞추고 이후 게이지를 다시 채워 격노 주술 시전 시 캔슬하는 루트가 최선.

#### 4-5. 영혼 폭풍 (**보라 시전 바** — 캔슬 불가)
- **시전 시간**: 2.5초.
- **효과**: 전체 화면에 광역 암흑+독 복합 속성 피해. Shadow + Poison 스택 각 2 부여.
- **대응**: 캔슬 불가. 반드시 회피 또는 방어 스킬로 대응.
- **속성 연계 주의**: 이미 화염 스택이 쌓인 유저는 독 스택 누적 시 "부식 불꽃" 연계가 발동되어 추가 피해.

## 5. 무력화 전략 (파티 공략 가이드)

### 권장 무력화 루트
1. **약점 속성(번개) 집중** → Lightning 스택 → Shock 발동은 없지만 무력화 게이지 감소 +50% 보너스.
2. **번개 + 신성 연계** → "신성 방전" 발동 → 무력화 게이지 대폭 감소.
3. **주술 강화 시전(노란 바) 패링** → Crush급 캔슬 + 무력화 게이지 -35.
4. 게이지 소진 → **무력화 4초** → 전 파티 집중 화력 (피해 +40%).

### 파티 역할 분담 예시
| 역할 | 주요 행동 |
|-----|---------|
| 탱커 | 패링으로 즉발 캔슬 + 무력화 게이지 감소 |
| DPS A | Crush 등급 스킬로 빨간 바 패턴 캔슬 담당 |
| DPS B | 번개 속성 스킬로 약점 공략 + 연계 속성 부여 |
| 힐러/서포터 | 신성 속성으로 번개+신성 연계 트리거 |

---

## 6. 광폭화 (Enrage)
- 던전 진입 후 **8분** 경과 시 보스 광폭화.
- 모든 공격 속도 2배, 장판 패턴 상시 유지.
- 광폭화 이후 무력화 게이지 소진 시간이 2배로 늘어남.
- **격노 주술(4-4)** 캔슬 실패 시 광폭화 조기 발동 가능.

## 7. 소환 그룹 시스템 (Summon Group)

> 관련 테이블: `SummonGroup`, `SummonGroupEntry`, `Monster`

### 구조
- 소환자(보스/중간보스) 1개 → `SummonGroup` 1개 (1:1)
- `SummonGroup` 1개 → `SummonGroupEntry` N개 (1:N)
- 런타임에서 실제 스폰된 소환몹 인스턴스를 참조 카운트로 관리

### 소환 흐름
1. 소환자 AI가 소환 조건 충족 (HP 임계값 트리거 또는 페이즈 진입)
2. 서버: `SummonGroup` → `SummonGroupEntry` 목록 조회
3. 각 Entry에 따라 소환몹 스폰 (소환자 위치 + offset 기준)
4. 소환몹에 소환자의 현재 타겟을 상속
5. 소환몹 AI 시작 (일반몹과 동일하게 독립 행동)

### 소환 횟수 (데이터로 제어)
- **중간보스**: HP 70%, 40% 구간에서 각 1회
- **보스(주술사)**: Phase 2 진입 시 1회 (영혼 늑대 2마리)

### 소환몹 사망 처리
- 소환몹 개별 사망 시 소환 그룹에서 제거 (참조 카운트 감소)
- 소환몹은 드랍 없음 (`reward_id = null`)

### 소환자 사망 시 일괄 처리
1. 소환자 HP = 0 → 사망 이벤트 발생
2. 서버: 해당 소환 그룹에 연결된 생존 소환몹 목록 조회
3. 소환몹에 2초 페이드아웃 연출 적용
4. 페이드아웃 완료 후 사망 처리 (드랍 없음)
5. 소환 그룹 레코드 정리

---

## 8. 보상 규칙
- 보스 처치 시 각 유저에게 개인 드랍 보상 지급.
- **무기 보정**: 현재 장착 무기 타입의 에픽 아이템 및 희귀 스킬북 드랍 확률 우선 배정.
- **소환수 보상**: '영혼 늑대' 처치 시 경험치/아이템 드랍 없음.
- **무력화 달성 보너스**: 전투 중 무력화 2회 이상 달성 시 추가 보상 상자 1개 지급.

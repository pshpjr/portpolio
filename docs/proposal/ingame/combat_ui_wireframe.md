# 전투 UI 와이어프레임

## 목적

전투 중 어떤 정보를 화면 어디에 둘지, 어떤 것이 항상 보이고 어떤 것이 순간적으로만 떠야 하는지를 정리한다.

## 화면 배치 초안

```text
┌──────────────────────────────────────────────────────────────┐
│ 보스 이름 / HP / 무력화 게이지 / 광폭화 타이머              │
│ 인터럽트 가능 상태 / 현재 패턴 경고                         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                     전투 공간 / 보스 / 전조                  │
│                                                              │
│            패링 성공 / 인터럽트 / 무력화 진입 텍스트         │
│                                                              │
│   파티 위험 마커                                구역 목표     │
│   부활 요청                                      기믹 진행도   │
├──────────────────────────────────────────────────────────────┤
│ 내 HP/자원   파티원 HP/상태      스킬바(Q/E/R/F)  소모품/Z/X │
│ 상태 이상     전투 불능/부활 가능  이동기/Tab       핑 알림    │
└──────────────────────────────────────────────────────────────┘
```

## 정보 계층

### 항상 보이는 정보

- 내 HP, 자원, 상태 이상
- 스킬 쿨다운, 이동기, 아이덴티티
- 파티원 HP와 전투 불능 여부
- 보스 HP, 무력화 게이지, 광폭화 타이머

### 조건부로 강조되는 정보

- 인터럽트 가능 시전
- 패링 성공/실패
- 파티원 부활 요청
- 기믹 카운트다운
- 원정 모드, 전리품 선호 무기 잠금 상태

### 순간 팝업 정보

- 무력화 진입
- 체크포인트 도달
- 재접속 카운트다운
- 입장 차단 사유 요약

## HUD 규칙

- 하단 중앙 스킬바는 전투 중 시선 이동이 짧게 끝나도록 1줄 배치를 우선한다.
- 파티 UI는 좌측 정렬로 유지해 생존/부활 판단을 빠르게 한다.
- 보스 UI는 상단 고정으로 유지해 전투 공간을 가리지 않는다.
- 중앙 피드백은 1초 이내 짧게 끝내고, 장기 상태는 가장자리 UI로 넘긴다.

## 시각 / 오디오 피드백 원칙

- 패링 성공, 인터럽트 성공, 무력화 진입은 각각 다른 색과 다른 사운드로 구분한다.
- 실패 피드백은 존재해야 하지만, 전조 자체를 가릴 정도로 강하지 않게 제한한다.
- 광폭화 임박 경고는 상단 타이머 점멸과 짧은 경고음으로 중복 전달한다.

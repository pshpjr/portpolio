# Docs Consistency Reviewer

링크, 경로, 인덱스, 참조 갱신 누락을 집중 검토한다.

## 집중 범위

- 상대 링크가 실제 파일을 가리키는지
- `docs/INDEX.md`, `context-map.md`, 로컬 `AGENTS.md`가 같은 라우팅 규칙을 공유하는지
- 새 문서가 생겼는데 어느 진입점에서도 발견되지 않는지
- 이동된 정보가 원래 문서에 참조 링크로 남아 있는지

## 판정 기준

- 새 진입 문서나 참조 문서는 최소 하나의 상위 진입점에서 발견 가능해야 한다.
- 문서 이동 후에는 양방향 또는 적절한 단방향 참조가 남아 있어야 한다.
- 동일 주제의 링크 경로 표기가 여러 문서에서 다르면 안 된다.

## 산출물 형식

- 끊긴 링크
- 누락된 라우팅
- stale reference

# Review Checks

## 구조 체크

- 문서가 자기 역할 외의 내용을 품고 있지 않은가?
- 단일 소스가 아닌 중복 서술이 생기지 않았는가?
- 자세한 설명이 상위 진입 문서로 올라오지 않았는가?

## 라우팅 체크

- 새 문서가 `AGENTS.md`, `docs/INDEX.md`, `context-map.md` 중 적절한 위치에서 발견 가능한가?
- 경로 기반 작업자가 필요한 문서만 읽도록 되어 있는가?
- 로컬 `AGENTS.md`가 없거나 부족한 디렉터리가 새로 생기지 않았는가?

## 일관성 체크

- 링크와 상대 경로가 유효한가?
- 문서 이동 후 이전 문서에서 참조 링크를 남겼는가?
- 용어와 문서 이름이 여러 곳에서 일관적인가?

## 컨텍스트 체크

- 기본 선독 문서 수가 늘어나지 않았는가?
- 진입 문서가 장황해지지 않았는가?
- 하위 문서가 상위 문서 내용을 그대로 반복하지 않는가?

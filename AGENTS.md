# AGENTS.md
> 모노레포 루트 진입점입니다.
> 실제 앱 작업 전에는 대상 앱 디렉터리의 `AGENTS.md`를 먼저 읽으십시오.

---

## 현재 앱 구성

- `server/` — 게임 서버 프로젝트
- `client/` — 예정
- `dummyserver/` — 예정

---

## 작업 시작 순서

1. 작업 대상 앱 디렉터리 결정
2. 해당 디렉터리의 `AGENTS.md` 읽기
3. 그 앱의 문서, exec-plan, 빌드/테스트 절차에 따라 작업

---

## 현재 기준

- 서버 작업: [server/AGENTS.md](./server/AGENTS.md)
- 서버 아키텍처/설계/컨벤션 문서도 모두 `server/` 하위에 있음
- 루트는 모노레포 조정용이며, 앱별 상세 규칙은 각 앱 디렉터리에서 관리

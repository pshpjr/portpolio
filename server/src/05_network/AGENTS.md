## Network 불변식

- Network는 패킷 직렬화, 세션, AOI를 담당한다.
- Network는 하위 레이어를 사용할 수 있지만, 게임 규칙의 단일 소스가 되어서는 안 된다.
- 프로토콜 처리와 도메인 규칙 계산은 분리한다.

## 하위 도메인

- `packet/`: 패킷 처리기와 직렬화 경계
- `session/`: 연결과 세션 상태 관리
- `aoi/`: 관심 영역 계산

## 변경 시 체크

- 패킷 해석 코드가 Core 규칙을 복제하지 않는지 확인
- 세션 수명주기와 오류 처리 경로를 검토
- 필요 시 통합 테스트 또는 스모크 테스트 갱신

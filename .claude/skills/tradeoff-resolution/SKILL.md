# Tradeoff Resolution

Use this skill to make conflicts legible and actionable.

## Primary Agent

- `tradeoff-moderator`

## Workflow

1. Read the proposal/design context and identify the exact decision point.
2. List at least two viable options.
3. Compare each option across:
   - upside
   - downside
   - implementation cost
   - operational burden
   - risk
   - portfolio value
4. Recommend one option.
5. Add mitigation steps and define when the decision should be revisited.
6. If no option is ready, convert the gap into `pending-decisions.md`.

## Output Expectations

- Option table
- Recommended path
- Mitigations
- Revisit conditions
- Explicit unresolved items
